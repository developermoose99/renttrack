# 🏠 RentTrack — Property Manager

A fully cloud-synced tenant and payment tracking app, hosted on GitHub Pages and powered by Supabase.

## Live App
👉 [Open RentTrack](https://YOUR-USERNAME.github.io/renttrack)

## Features
- 🔐 Login accounts with Admin / Viewer roles
- ☁️ Real-time cloud database (Supabase)
- 📊 Interactive dashboard with charts
- 💳 Tenant & payment tracking
- 📱 Works on any device

## Tech
- Pure HTML/CSS/JS — no build step needed
- Supabase for auth + database
- GitHub Pages for hosting
